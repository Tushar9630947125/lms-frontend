# Lms Frontend2

### Setup instruction

1. Clone the project

   ```
   git clone https://github.com/Tushar9630947125/lms-frontend.git
   ```

2. Move into the directory

   ```
   cd lms-frontend
   ```

3. install dependencies

   ```
   npm i
   ```

4. Run the server
   ```
   npm run dev
   ```
